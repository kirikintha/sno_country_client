##Summary

The Sno Country Client module uses a feed from ski.bluetent.com and queries a particular path for snow weather data.

##How It works

Enter in the url you wish to pull from, the key you wish to pull from, and the period of time you wish to pull from, and then you can use the supplied widgets and blocks for pulling that data and themeing it.
